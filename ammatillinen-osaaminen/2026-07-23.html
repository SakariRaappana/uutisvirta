Microsoft Foundry esitteli uuden työkalupakin agenttien rakentamiseen, mikä yksinkertaistaa agenttien toteutusta ja toimintaa. Samana päivänä Microsoft julkaisi Agent Framework Harness -työkalun, joka mahdollistaa agenttien kehittämisen vakaalla ja kattavalla alustalla.

Building Agents that Act on Your Behalf with Toolboxes in Foundry

Microsoft Foundry on julkaissut uudet työkalupakit, joiden avulla kehittäjät voivat rakentaa agentteja, jotka toimivat käyttäjien puolesta. Työkalupakit ovat osa Foundry Agent Serviceä ja ne hyödyntävät Microsoftin Communicative Protocolia (MCP) helpottaakseen komponenttien välistä viestintää. Tämä työkalu tarjoaa kehittäjille valmiit komponentit, mikä vähentää koodaustarvetta ja nopeuttaa kehitysprosesseja. Työkalupakit ovat osa suurta pyrkimystä helpottaa tekoälyagenttien rakentamista ja käyttöönottoa monimutkaisissa ympäristöissä. Lue alkuperäinen →

The Microsoft Agent Framework Harness is now released

Microsoft on julkaissut Agent Framework Harness -työkalun, joka tarjoaa vankan ja virtaviivaisen alustan tekoälyagenttien kehittämiseen. Tämä julkaisu sisältää kaikki tarvittavat työkalut agenttien tehokkaaseen rakentamiseen ja helpottaa .NET- ja Python-ympäristöjen integrointia kehitystyöhön. Uudessa versiossa keskitytään vakauttamaan aiempia kehitystöitä, mikä mahdollistaa laajemman sovellusalueen hyödyntämisen. Harness tekee agenttien kehittämisestä helpompaa ja parantaa niiden suorituskykyä ja luotettavuutta. Lue alkuperäinen →
