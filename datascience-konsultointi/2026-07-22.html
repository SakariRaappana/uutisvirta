Tänään Microsoft julkisti Microsoft Agent Framework Harnessin, joka tarjoaa vakaan alustan tekoälyagenttien kehittämiselle. Lisäksi Microsoft ja Mistral laajensivat kumppanuuttaan tarjotakseen edistyneitä, hallittavissa olevia tekoälyratkaisuja yrityksille ja säännellyille toimialoille.

Tärkeimmät uutiset

The Microsoft Agent Framework Harness is now released

Microsoft on julkaissut Microsoft Agent Framework Harnessin. Tämä uusi kehys mahdollistaa tekoälyagenttien rakentamisen tehokasta kontekstinhallintaa, suunnittelua, muistinhallintaa ja seurantaa varten. Kehys tukee sekä Python- että .NET-ohjelmointikieliä, ja se toimii agentin arkkitehtuurin tukirankana, mahdollistaen kielimallien muuttamisen toimiviksi agenteiksi. Uuden julkaisun myötä kehittäjät saavat käyttöönsä valmiit työkalut tekniikoiden, kuten silmukoiden ja hyväksyntöjen, toteuttamiseen. Microsoft and Mistral expand strategic partnership

Microsoft ja Mistral ovat laajentaneet strategista kumppanuuttaan, keskittyen säänneltyjen toimialojen ja yritysten tarpeisiin tarjota hallittavissa olevia tekoälyratkaisuja. Tämän laajennuksen kautta he aikovat mahdollistaa tekoälyn uusimmat mahdollisuudet, joiden avulla organisaatiot voivat säilyttää kontrollin ja dataturvan. Mistral tuo kokemuksensa tekoälyn kehittämiseen, kun taas Microsoft tarjoaa infrastruktuurinsa ja asiantuntemuksensa laajassa mittakaavassa. Tämä suuntaus on erityisen tärkeä säännellyissä ympäristöissä, kuten finanssialalla, joissa compliance-vaatimukset ovat tiukkoja. Lue alkuperäinen →
